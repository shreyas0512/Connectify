# Getting Started
## Dependencies

* react: 18.2.0
* react-dom : 18.2.0
* firebase : 9.17.2
* react-router-dom : 6.9.0
* tailwindcss : 3.2.7
## Installation

1. Clone the repo
   ```sh
   git clone -b posts https://github.com/shreyas0512/carestack
   ```
2. Install NPM packages
   ```sh
   npm i
   ```
3. Start the Development Server
   ```sh
   npm run dev
   ```
